# 🛡️ Neural Fraud Shield

A Machine Learning based Financial Fraud Detection System that combines Deep Neural Networks (DNN), XGBoost, and Explainable AI techniques (SHAP & LIME) to detect fraudulent credit card transactions.

## 🚀 Features

* Deep Neural Network (TensorFlow/Keras)
* XGBoost Baseline Model
* SHAP Explainability
* LIME Explainability
* Fraud Risk Analysis Dashboard
* Real-Time Prediction API
* Interactive Web Interface
* Class Imbalance Handling using SMOTE

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* XGBoost
* Scikit-Learn
* Flask
* SHAP
* LIME
* HTML, CSS, JavaScript

## 📊 Dataset

Credit Card Fraud Detection Dataset from Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Place the dataset as:

```text
data/creditcard.csv
```
