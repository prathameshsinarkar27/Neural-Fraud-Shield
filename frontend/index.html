Model Showdown: DNN vs XGBoost
The Reality of Tabular Data: While the DNN utilizes a complex multi-layer architecture, XGBoost significantly outperforms the DNN on this dataset. This is common in machine learning: gradient-boosted decision trees (XGBoost) naturally excel at tabular, structured data with extreme class imbalances. XGBoost achieves a vastly superior F1 Score (0.75 vs 0.42) because it yields far fewer False Positives, making it the more practical model for a real-world production environment.

Deep Learning Exploration: How the baseline DNN was trained using Backpropagation
Why visualize the DNN? While XGBoost is our winning production model, tree-based algorithms build discrete trees rather than updating continuous weights. We visualize the DNN here to demonstrate the mechanics of gradient descent, backpropagation, and epoch-based learning in high-dimensional PCA space.
Training Process: How Backpropagation Works
1
Forward Pass
Input data flows through Dense → BatchNorm → ReLU → Dropout layers to produce a fraud probability prediction.
2
Loss Computation
Binary Cross-Entropy measures the error between predicted probability and actual label (fraud/legit). Loss = -[y·log(ŷ) + (1-y)·log(1-ŷ)]
3
Backward Pass
Backpropagation computes ∂Loss/∂weights for each layer using the Chain Rule, propagating gradients from output back to input.
4
Weight Update (Adam)
Adam optimizer updates weights using adaptive learning rates based on 1st moment (mean) and 2nd moment (variance) of gradients.
